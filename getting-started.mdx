## Setting Up Your Account

Getting started with Ahmad Assayyaaf is straightforward. Follow these steps to create your account and set up your first workspace.

<Steps>
  <Step title="Create Account" icon="user">
    Visit our website and sign up for a new account. Choose between personal and team plans based on your needs.
    
    <Callout kind="info">New users receive a 14-day free trial to explore all features.</Callout>
    
    <Expandable title="Account Verification" default-open="false">
      After signing up, check your email for a verification link. Click the link to activate your account and proceed to the dashboard.
    </Expandable>
  </Step>
  <Step title="Configure Profile" icon="settings">
    Complete your profile by adding your name, profile picture, and preferences.
    
    <Tabs>
      <Tab title="Basic Settings" icon="user">
        Set your display name, timezone, and language preferences.
        
        <CodeGroup tabs="JSON,cURL">
          ```json
{
  "name": "John Doe",
  "timezone": "America/New_York",
  "language": "en-US"
}
```
          ```bash
curl -X PUT https://api.ahmadassayyaaf.com/v1/users/profile \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "Jane Smith", "timezone": "Europe/London"}'
```
        </CodeGroup>
      </Tab>
      <Tab title="Advanced Settings" icon="sliders">
        Configure notification settings and privacy options.
        
        <Expandable title="Notification Preferences" default-open="false">Choose how you want to receive updates about workspace activity and AI tool usage.</Expandable>
      </Tab>
    </Tabs>
  </Step>
  <Step title="Create Workspace" icon="folder">
    Set up your first workspace to organize your projects and documents.
    
    <CodeGroup tabs="JavaScript,Python">
      ```javascript
const workspace = await ahmad.createWorkspace({
  name: 'My First Workspace',
  description: 'Personal productivity space',
  settings: { visibility: 'private' }
});
console.log(`Workspace created: ${workspace.id}`);
```
      ```python
workspace = client.create_workspace(
    name='Team Project',
    description='Collaborative workspace for marketing team',
    visibility='shared'
)
print(f'Created workspace: {workspace.id}')
```
    </CodeGroup>
    
    <Callout kind="tip">Start with a descriptive name and consider your collaboration needs when setting visibility options.</Callout>
  </Step>
</Steps>

## First AI Search

Try your first AI-powered search to see how the platform understands context and intent.

<Expandable title="Search Examples" default-open="true">
  Search for specific topics, ask questions, or use natural language queries. The AI understands context and can find relevant information even when you don't remember exact terms.
  
  <CodeGroup tabs="Web Interface,API">
    ```javascript
document.getElementById('search-input').addEventListener('input', async (e) => {
  const results = await ahmad.search({
    query: e.target.value,
    workspace: currentWorkspace,
    filters: { type: 'document' }
  });
  displayResults(results);
});
```
    ```bash
curl "https://api.ahmadassayyaaf.com/v1/search?q=project%20deadlines&workspace=team-alpha" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
  </CodeGroup>

## Connecting with Team Members

Invite colleagues to collaborate in your workspace.

<Callout kind="info">Team collaboration features are available on all plans and allow real-time editing and commenting.</Callout>

<CodeGroup tabs="Invitation API,CLI Tool">
  ```javascript
const invitation = await ahmad.inviteUser({
  email: 'colleague@company.com',
  workspace: 'team-workspace',
  role: 'editor',
  message: 'Join our collaborative workspace!'
});
console.log(`Invitation sent to ${invitation.email}`);
```
  ```bash
ahmad workspace invite --email team@company.com --role contributor --workspace marketing
```
</CodeGroup>
