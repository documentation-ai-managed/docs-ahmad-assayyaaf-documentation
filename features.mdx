## AI-Powered Search

Our intelligent search feature goes beyond simple keyword matching to understand context and intent, helping you find exactly what you need.

<Tabs>
  <Tab title="Natural Language Search" icon="search">
    Ask questions in plain English and get relevant results from across your workspace.
    
    <CodeGroup tabs="JavaScript,Python">
      ```javascript
const results = await ahmad.search({
  query: 'What are the deadlines for Q4 projects?',
  workspace: 'company-workspace',
  sortBy: 'relevance'
});
console.log(`Found ${results.length} relevant items`);
```
      ```python
search_results = client.natural_language_search(
    query="Show me all documents about AI integration",
    workspace_id="tech-team",
    include_attachments=True
)
for result in search_results:
    print(f"{result.title}: {result.snippet}")
```
    </CodeGroup>
    
    <Callout kind="tip">Use specific questions for better results, and the AI learns from your interactions to improve future searches.</Callout>
  </Tab>
  <Tab title="Advanced Filters" icon="filter">
    Narrow down results with powerful filtering options.
    
    <Expandable title="Filter Options" default-open="true">
      | Filter Type | Description | Example |
      |-------------|-------------|---------|
      | Date Range | Limit to specific time periods | `last-week`, `this-month` |
      | Content Type | Documents, notes, or images | `type:document` |
      | Author | Content created by specific users | `author:john.doe` |
      | Tags | Items with specific tags | `tags:urgent,review` |
    </Expandable>
  </Tab>
</Tabs>

<Steps>
  <Step title="Start Writing" icon="keyboard">
    Open a new document and begin typing. The AI assistant will provide suggestions as you write.
    
    <Expandable title="Writing Modes" default-open="false">
      Choose from different writing modes: creative, professional, concise, or conversational.
    </Expandable>
  </Step>
  <Step title="Use AI Suggestions" icon="cpu">
    Accept, modify, or ignore AI-generated suggestions for improved content.
    
    <CodeGroup tabs="API Example,cURL">
      ```javascript
const suggestions = await ahmad.getWritingSuggestions({
  text: 'The company is launching a new product',
  mode: 'professional',
  tone: 'enthusiastic'
});
console.log(suggestions);
```
      ```bash
curl -X POST https://api.ahmadassayyaaf.com/v1/writing/suggestions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"text": "Our team completed the project", "style": "formal"}'
```
    </CodeGroup>
  </Step>
  <Step title="Generate Content" icon="plus">
    Use prompts to generate entire sections or documents from scratch.
    
    <Callout kind="info">Content generation works best with specific, detailed prompts that include your requirements and target audience.</Callout>
  </Step>
</Steps>

Organize your thoughts and ideas with smart note-taking features that automatically categorize and connect related content.

<ExpandableGroup>
  <Expandable title="Auto-Categorization" default-open="true">
    Notes are automatically tagged and categorized based on content analysis.
    
    <Tabs>
      <Tab title="Manual Tags" icon="tag">Add your own tags for custom organization.</Tab>

    </Tabs>
  </Expandable>
  <Expandable title="Note Linking" default-open="false">
    Connect related notes and documents to create a knowledge network.
    
    <CodeGroup tabs="Link Creation,CLI">
      ```javascript
const link = await ahmad.createNoteLink({
  sourceNote: 'meeting-notes-2024-01',
  targetNote: 'project-timeline',
  relationship: 'related-to'
});
console.log(`Link created: ${link.id}`);
```
      ```bash
ahmad notes link --source meeting-123 --target project-plan --type "supports"
```
    </CodeGroup>
  </Expandable>
